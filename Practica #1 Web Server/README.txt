				PRACTICA #1 SERVIDOR WEB

INSTRUCCIONES DE USO 

1.- Acceder al directorio donde se encuentran todos los archivos de la practica

2.- Compilar los archivos con el comando “make all”

3.- Ejecutar el archivo webserver.c “./webserver -p 8080 -vvvv”

4.- Desde un navegador web (Safari, Firefox, Chrome etc.) acceder a la dirección del           servidor 127.0.0.1:8080, se mostrara la pagina principal del servidor.

5.- Para solicitar un archivo es necesarios incluir la dirección anterior e incluir el nombre del archivo con su extensión. 
	ejemplo:
		127.0.0.1:8080/imagen1.jpg
